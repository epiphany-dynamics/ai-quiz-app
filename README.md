# AI Readiness Quiz

An interactive assessment tool that helps business owners evaluate their organization's readiness for AI adoption. Gamified experience with scoring, milestone celebrations, and personalized results.

## What It Does

- **Multi-Track Assessment** — Different question paths based on business type and goals
- **Gamified Experience** — Progress tracking, milestone celebrations, confetti animations, and sound effects
- **Smart Scoring** — Weighted scoring engine that categorizes results into actionable tiers
- **Results Dashboard** — Score visualization with ring chart and personalized recommendations
- **Share & Compare** — Shareable result links so teams can compare scores
- **Email Capture** — Optional opt-in for detailed follow-up resources
- **Responsive Design** — Works on desktop and mobile

## Tech Stack

- **React 18** — Component-based UI with context state management
- **TypeScript** — Full type safety across the quiz engine and components
- **Vite** — Fast build tooling
- **Tailwind CSS** — Utility-first styling
- **Supabase** — Backend for result storage and email capture
- **Framer Motion** — Animations and transitions
